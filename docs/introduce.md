1. 미술 작품 중개 플랫폼 "Artistry"
=> 아마추어 일러스트레이터들을 위한 그림 재능 중개 플랫폼 "ARTISTRY"입니다.
개인 SNS나 블로그에 의존하던 기존의 파편화된 홍보 및 거래 방식을 개선하고, 작가와 소비자가 신뢰를 바탕으로 소통하고 거래할 수 있는 전문화된 공간을 제공하기 위해 기획되었습니다.
담당 역할: 팀장 (Team Leader) 및 Full-Stack 개발 (Frontend/Backend) 
주요 담당 기능: SNS 로그인 API 연동, WebSocket 기반 실시간 채팅 기능 구현, 프로젝트 총괄 및 통합 테스트 
주요 기능:
작가별 포트폴리오 등록 및 관리 
WebSocket을 활용한 작가-소비자 간 1:1 실시간 소통 (채팅) 
Crawling을 통한 작가 및 작품 데이터 수집 
JSP/Servlet 및 AJAX 기반의 '좋아요(찜하기)' 기능 
적용 기술 (Tech Stack):
Frontend: JSP, jQuery, Webflow (애니메이션) 
Backend: JSP/Servlet 
Real-time: WebSocket 
APIs: SNS Login (Kakao, Naver, Google), Payment API (아임포트), 주소 API 
Database: DB (설계 및 구축)

2. 챗봇을 활용한 LLM 투자성향 분석 큐레이션 서비스 "InvestGenius"
=> LLM(LLaMA3, ChatGPT3.5)을 기반으로 개인 투자 성향을 분석하고 , 맞춤형 투자 포트폴리오를 추천하는 대화형 AI 챗봇 서비스입니다.
코로나 이후 급증한 개인 투자자들이 전문 지식 없이도 비대면 환경에서 실시간 시장 변화가 반영된  투자 전략을 제안받을 수 있도록 기획되었습니다.
담당 역할: 팀장 (Team Leader) 및 Full-Stack 개발 (Frontend, Backend) 
주요 기능:
LLM을 활용한 사용자 투자 성향(공격, 중립, 방어형) 분석 
분석 결과에 따른 맞춤형 투자 포트폴리오 자동 제안 
실시간 국내/외 주식 데이터 및 기업 재무제표 조회 및 시각화 
적용 기술 (Tech Stack):
Frontend: React.js (사용자 친화적 인터페이스 및 차트 시각화) 
Backend: Spring Boot (JPA), Python (Flask) (REST API 및 데이터 파이프라인) 
LLM: LLaMA3, ChatGPT3.5 (파인튜닝 및 프롬프트 엔지니어링) 
Database: MySQL
